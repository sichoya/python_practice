🚀 FastAPI 실행
터미널(zsh)에서 /프로젝트_폴더로 이동한 후, 다음 명령어를 실행하여 **main.py**를 서버의 시작점으로 지정합니다.

uvicorn main:app --reload --port 8000

🌐 브라우저에서 확인
서버가 성공적으로 실행되면, 아래 URL을 브라우저에 입력하여 API의 동작을 테스트하고, FastAPI가 자동으로 생성한 문서를 확인할 수 있습니다.

전체 메뉴: http://127.0.0.1:8000/menu

특정 메뉴: http://127.0.0.1:8000/menu/1

문서 자동 생성:

Swagger UI → http://127.0.0.1:8000/docs

ReDoc → http://127.0.0.1:8000/redoc
